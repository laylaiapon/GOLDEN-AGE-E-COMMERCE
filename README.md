## 📌 Gerenciador de Links.
*  Nome do Projeto- loja de e-commerce.
*  Descrição do Projeto- Este é um projeto de um site de e-commerce e apresentação para a GOLDEN AGE, uma marca fictícia de relódios de luxo.

## 📌Por que implementar uma loja de e-commerce? 
* Implementamos o e-commerce da GOLDEN AGE para democratizar o acesso ao luxo sem perder a exclusividade. O site funciona como nossa vitrine principal, onde o design dourado e a interatividade do sistema de login criam uma jornada de compra sofisticada, segura e disponível em tempo integral.

## 📌 Principais aprendizados.
* Enfrentei um desafio ao precisar usar google maps dentro do meu código com a localização da loja.
* Utilizei as documentações para ajudar no código.
