Este é um projeto de um site de e-commerce e apresentação para a GOLDEN AGE, uma marca fictícia de relódios de luxo.

SOBRE O PROJETO:

O objetivodeste projeto foi desenvolver uma interface moderna, focada na experiência do usuário.
Página Inicial: vitrine de procutos com cards.
Sobre Nós: História da marca com design em degradê azul.
Localização: Integração com google Maps Para visualização.

TECNOLOGIAS UTILIZADAS:

Para a construção deste projeto, foram utilizadas as seguintes tecnologias.

HTML5: Estruturação semântica do conteúdo
CSS: Estilização avançada
JAVASCRIPT: Lógica de direcionamento do botão
GOOGLE MAPS: Integração de Geolocalização
